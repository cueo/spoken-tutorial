Please note, GNU Bash version 4 recommended practice tutorial. Let us first learn pass argument function usage. Let open file 'function_(underscore) parameters.sh' . This shebang line. Our function name say_(underscore)welcome. Open curly bracket opens function definition. '$1' (Dollar one) first positional parameter. '$2' (Dollar two) second positional parameter. Close curly bracket closes function definition. Here, function 'say_welcome' called arguments. The syntax is- function name i.e. "say_welcome" followed arguments within double quotes, i.e. "Bash" "learning". In similar manner, I call function different set arguments. So, I say_welcome space within double quotes functions space within double quotes Bash. Save file go terminal. Type: chmod space plus x space function underscore parameters dot sh. Press Enter. Type: dot slash function underscore parameters dot sh. Press Enter. We see positional parameters substituted arguments passed function. Dollar 1 ($1) substituted string "Bash" Dollar 2 ($2) "learning". Then again, Dollar 1 ($1) substituted "functions in" Dollar 2 ($2) "Bash". In Bash, variables declared local variables global variables. Local variable: * Its value valid within function defined. * Local variables declared using keyword local. Global variable: The value global variable accessed throughout Bash script. Let us learn 2 ways declare variable within function. Let open file named 'function_(underscore)local.sh'. This shebang line. Function name say_(underscore) hello. Here, variable first_name declared keyword local means value valid within function say_hello only. A variable declared without keyword treated global variable. So, variable last_name accessed throughout script. In echo line, display value variables first_name, middle_name last_name. After this, close function. Now, here, variable middle_name declared without keyword. So, value global throughout script. Once again, call function here. We pass two arguments function call namely- “Pratik” “Patil”. These echo statements display value variables- $first_name, $middle_name $last_name. Please keep mind variable first_name local variable. Save file go terminal. Type: chmod space plus x space function underscore local dot sh. Press Enter. Type: dot slash function underscore local dot sh Press Enter. The first line output displays message "Hello Pratik K Patil". Here, variable first_name contains value "Pratik" local means value limited function. Now, let us see local variable behaves outside function. Here, nothing displayed first_name. This value first_name local function.
