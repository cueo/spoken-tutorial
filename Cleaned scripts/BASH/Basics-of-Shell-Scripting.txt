For tutorial, I using: * Ubuntu Linux 12.04 Operating System * GNU Bash version 4.1.10 Please NoteGNU Bash version 4 recommended practice. Let us start introduction variables. Bash variables provide temporary storage information. These variables used within lifespan program. There two types variables: System variables User defined variables . System variables: These created maintained Linux Bash Shell itself. They defined capital letters. Commonly used system variables are- * BASH_VERSION * HOSTNAME * HOME etc. Let us open terminal pressing Ctrl, Alt T keys simultaneously keyboard. Now, type: set press Enter. This display system variables. Alternately, type env printenv view system variables. Let clear prompt. Now, type: echo space within double quotes dollar sign HOSTNAME press Enter. The hostname system displayed. Now let's find full path homedirectory. Type: echo space within double quotes dollar sign HOME(in capital). Press Enter. The full path user's home directory displayed. Now, type: echo space within double quotes HOME (in capital). Press Enter. This display "HOME", value HOME variable. So, necessary use dollar sign( $) beginning every variable display value. Let us switch back slides. User Defined Variables: These variables created maintained users. It always good idea avoid uppercase names user defined variables. This makes easy differentiate user defined system variables. Switch back terminal. Type: username equal sign sunita Please note blank space username, equal sign sunita. Now, press Enter. To display value variable username, Type: echo space within double quotes dollar sign username press Enter. This display sunita terminal. The value variable unset. Let us switch back slides. unset- value variable unset using unset command. The syntax is- unset variablename Let's use previous example username variable. Switch Terminal. Now type: unset space username, press Enter. Let us check. Type: echo space within double quotes dollar sign username press Enter. Nothing displayed terminal. This means value variable username removed. Now switch back slide. Global local variables: In Shell script, user defined variables declared globally locally. By default, variables global. This means, values remain inside outside function. Let us learn declare variables globally locally. Switch terminal . Type: gedit space g_(underscore)variable.sh space & (ampersand sign). gedit text editor g_(underscore) variable.sh file name '&' (ampersand) used free prompt. Press Enter. Type code shown here, "g_(underscore)variable.sh" file. Let explain code now. The first line hash exclamation symbol shebang bang line. username=sunita userdefined variable declared globally. echo display string "outside function:" dollar username print value variable username. This define function BASH script. We discuss functions detail, later tutorials. This body function. Here, another message "inside function" displayed along value username. Here, call function. This code. Now let's execute it. Come back Terminal. Let clear prompt. First need make file executable. Type: chmod space plus x space g_(underscore)variable.sh, press Enter. Now, type: dot slash g_(Underscore)variable.sh Press Enter. Observe output. Outside function, username takes value sunita. Inside function also, username takes value sunita. This username declared globally outside function. Next, let us learn declare variable locally. Type: gedit space l_(Underscore)variable.sh space & (ampersand sign). Press Enter. Type code shown here, "l_(underscore)variable.sh" file. Let explain code. The code except extra line code inside function. Inside function block, line- local space username equals jack. This assigns new value variable username locally. Now switch Terminal. Let's make file executable typing chmod space plus x space l_variable.sh Press Enter. Type: dot slash l_variable.sh Press Enter. The output displayed. Outside function, username takes value sunita whereas inside function, username takes value jack. This username assigned value locally, within function. Now let us quickly see get user input via keyboard. The read command used accept input keyboard. It also used assign input value user defined variable. The syntax read command is- read space hyphen p space within double quotes PROMPT Please note PROMPT string waits user input. You may replace string. Now, switch terminal . Type: gedit space read.sh space & (ampersand sign) Press Enter. Type code shown "read.sh" file. Let explain code. In example, input given keyboard user. This bang line. Here -p displays prompt without newline takes input keyboard. The user input stored variable username. echo command displays message Hello name entered user via keyboard. So, let us execute program. Come back terminal. Type: chmod space plus x space read.sh press Enter. Type: dot slash read.sh press Enter. Here displayed Enter username: I type ashwini, press Enter. The message Hello ashwini displayed. ashwini assigned input value user defined variable username. Let us go back slides summarize.
