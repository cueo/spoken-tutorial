Welcome Spoken tutorial LibreOffice Calc – Formatting Data Calc. In tutorial, learn about: * Formatting Borders, background colors. * Formatting multiple lines text using Automatic Wrapping. * Merging Cells, Shrinking text fit cell. Here using GNU/Linux operating system LibreOffice Suite version 3.3.4. First, let us learn formatting borders LibreOffice Calc. Let us open “personal finance tracker.ods” file. Formatting borders done particular cell block cells. For example, let us format cells headings “Serial Number”, “Item”, “Cost”, “Spent”, ”Received”, ”Date” ”Account”. So, let us first click cell heading Serial Number, denoted “SN”. Now, hold left mouse button drag along cells containing headings. After selecting entire horizontal row containing headings, click “Borders” icon Formatting toolbar. A drop box opens containing several border styles. Click one styles want apply borders. I click last option. We see borders get formatted according selected style. Let us undo change. The selected cells still highlighted. Right-click selection choose “Format Cells” option. Now click “Borders” tab. You see options “Line arrangement”, “Line”, “Spacing contents” “Shadow style”. Displayed default settings Calc. But one change suit requirement. Under “User-defined”, see small preview window displays selection. I choose third option “Default” see reflected preview window. I also change “Style”, “Width” “Color”. Again, notice change preview window. Spacing contents “Synchronize” option checked. This means spacing applied margins. One uncheck change margin spacing per requirement. I change “Top” “Bottom” margins 1.4pt. I leave explore various Shadow styles, own. Click OK. This apply chosen style selected cells. After learning format borders, let us learn give background colors cells. In order give background colors cells, Calc provides option called “Background Color”, located Formatting toolbar. Now, let us see implemented. For example, let us give background color cells containing headings. So, let us first click cell contains heading Serial Number, denoted “SN”. Now hold left mouse button drag along cells containing headings. After selecting entire horizontal row containing headings, click “Background Color” option Formatting toolbar. A pop-up menu opens select color want apply background color. Let us click “Grey” color. You see cell background headings turns grey. Calc provides various options formatting multiple lines text. The first one using “Automatic Wrapping”. "Automatic Wrapping” allows user enter multiple lines text single cell. So, let us see implemented. Now, “personal finance tracker.ods” sheet, let us click empty cell. For example, let's click cell number “B12”. Now, right-click cell click “Format Cells” option. Now, click “Alignment” tab dialog box. At bottom dialog box, click “Wrap text automatically” option click OK button. Now, type “THIS IS A PERSONAL FINANCE TRACKER. IT IS VERY USEFUL”. You see multiple statements get wrapped single cell. Let's undo changes. After learning “Automatic Wrapping”, learn merge cells Calc. In “personal finance tracker.ods” file, want merge cells containing Serial Number heading “SN” corresponding items first click data entry '1' heading “SN”. Now, hold Shift key keyboard click cell corresponding item “Salary”. This highlights two cells merged. Next, click Format option menu bar click Merge Cells option. In sidebar pops up, click “Merge Cells” option. In order move contents cells single cell, click Yes option dialog box appears. You see selected cells get merged one contents also within merged cell. Now let us undo merging pressing CTRL+Z together. Next, learn shrink text fit cell. The font size data cell automatically adjusted fit cell. Let us learn it. Let's type text “This month January” cell referenced B14. You see text doesn't fit cell. In order shrink text fits, click cell referenced B14 first. Now click Format option menu bar click “Cells”. Alternately, right-click cell click “Format Cells”. You see “Format Cells” dialog-box opens. Click “Alignment” tab dialog-box. At bottom dialog box, click “Shrink fit cell size” check-box click OK button. You see entire text shrinks adjusts decreasing font size text fits cell referenced B14. Let's undo changes. This brings us end Spoken Tutorial LibreOffice Calc. To summarize, learned about: * Formatting Borders, background colors Calc. * Formatting multiple lines text using 'Automatic Wrapping'. * Merging Cells, Shrinking text fit cell.
